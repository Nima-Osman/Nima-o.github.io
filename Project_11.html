Neural Network Models for Object Recognition

Background

Artificial Intelligence (AI) technologies are integral to numerous industries, enabling advancements in domains such as autonomous vehicles, facial recognition, movement monitoring, and video security. Object detection capabilities are central to these applications.

This assignment focuses on developing a Neural Network using the CIFAR-10 image dataset for object recognition. The CIFAR-10 dataset consists of 60,000 32x32 color images across 10 object categories, with 6,000 images per category. It is a renowned dataset for visual recognition tasks, curated by Alex Krizhevsky, Vinod Nair, and Geoffrey Hinton.

Objective

To design and evaluate a Neural Network model for object recognition using the CIFAR-10 dataset. The task includes partitioning a validation set from the training data and assessing the model's efficacy.

Assignment Requirements

  • Dataset: Use the CIFAR-10 dataset, accessible via Kaggle or keras-datasets.
  • Data Partitioning: Outline the method for creating a validation set from the training data and provide metadata insights.
  • Validation Set Importance: Explain why maintaining a separate validation set is crucial.
  • Model Design:
    • Describe the architecture of the Neural Network.
    • Discuss the chosen activation function.
    • Explain the loss function used.
    • Detail the number of epochs used for training.
    • Provide insights into design strategies.
